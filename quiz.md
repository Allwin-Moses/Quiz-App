# 🎓 Quiz App

An interactive Quiz Application built using **ASP.NET MVC**, **C#**, **ADO.NET**, and **SQL Server**.

---

## 🚀 Features

- 📝 Multiple-choice quiz questions  
- ✅ Instant scoring and feedback  
- 🛠️ Manage questions easily  
- 📱 Responsive design for all devices  

---

## 💻 Tech Stack

- ASP.NET MVC  
- C#  
- ADO.NET  
- SQL Server
